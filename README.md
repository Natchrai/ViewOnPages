# ViewOn Pages

Natchrai가 개발하는 앱과 서비스를 소개하는 공식 랜딩페이지 모음입니다.

🌐 https://www.viewon.app

## 서비스

### ViewOn Link

두 대의 스마트폰을 QR 코드로 연결해, 한쪽 스마트폰의 카메라 화면을 다른 스마트폰에서 실시간으로 확인할 수 있는 앱입니다.

- QR 코드로 빠른 연결
- 같은 Wi‑Fi 환경에서 실시간 카메라 화면 확인
- 사진 및 동영상 저장
- 작은 글씨나 물체를 확대해서 볼 수 있는 돋보기 기능
- Android · iPhone 지원
- 한국어, 영어, 일본어, 아랍어, 독일어, 스페인어, 프랑스어 지원

- 소개 페이지: https://www.viewon.app/link/
- 사용 안내 영상: https://www.youtube.com/watch?v=0BVvc_ClXAg
- App Store: https://apps.apple.com/app/id6787559105
- Google Play: https://play.google.com/store/apps/details?id=com.natchrai.viewonlink

### Memosy

메모와 할 일을 간편하게 기록하고 관리하는 Android 앱입니다.

- 메모 및 할 일 관리
- 로컬 데이터 저장
- 백업 및 복원 지원

- 소개 페이지: https://www.viewon.app/memosy/
- Google Play: https://play.google.com/store/apps/details?id=com.natchrai.memosy
