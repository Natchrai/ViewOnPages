# HANDSOS × ViewOn Salon Link 제휴 제안 페이지

GitHub Pages에 바로 게시할 수 있는 정적 웹페이지입니다.

## 구성

- `index.html` — 페이지 내용
- `style.css` — 반응형 디자인
- `script.js` — 모바일 메뉴와 스크롤 애니메이션

## 게시 방법

### 저장소 루트에 올리는 경우

세 파일을 저장소 최상위 폴더에 업로드한 뒤 GitHub 저장소의
`Settings → Pages → Deploy from a branch`에서 대상 브랜치와 `/ (root)`를 선택합니다.

### 기존 사이트 저장소에 추가하는 경우

원하는 하위 폴더(예: `handsos-proposal`)를 만들고 세 파일을 업로드합니다.
게시 주소는 `https://계정명.github.io/저장소명/handsos-proposal/` 형식입니다.

현재 파일은 상대경로만 사용하므로 루트와 하위 폴더 어디에서든 동작합니다.
