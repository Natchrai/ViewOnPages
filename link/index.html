Main Features

폰 하나로는 어려웠던 장면을 더 쉽게

가까이 두기 어려운 곳, 내 눈으로 직접 보기 힘든 각도, 혼자 확인하기 어려운 자세까지 ViewOn Link로 확인하세요.

🔗

QR로 바로 연결

카메라용 휴대폰의 QR 코드를 Viewer용 휴대폰으로 스캔하면 두 기기가 바로 연결됩니다.

🔍

돋보기 모드

작은 글씨, 제품 번호, 전자부품처럼 가까이 봐야 하는 대상을 크게 확대해서 확인할 수 있습니다.

📷

사진 저장

확인 중인 장면을 사진으로 남겨 정비 기록, 제품 확인, 비교 자료로 보관할 수 있습니다.

🎥

영상 녹화

실시간 화면을 영상으로 저장해 작업 과정이나 확인 장면을 다시 볼 수 있습니다.

💡

어두운 곳 확인

가구 뒤, 침대 밑, 차량 하부처럼 빛이 부족한 곳도 더 편하게 확인할 수 있습니다.

⚙️

상황별 화질 선택

연결 환경에 맞춰 부드럽게 보거나 더 선명하게 보는 방식으로 화질을 선택할 수 있습니다.

Use Cases

생활 속 다양한 순간에 사용하세요

작아서 안 보일 땐 돋보기 모드

미세한 글씨, 제품 번호, 부품 표기를 크게 확대해서 확인하세요.

차량 하부 확인

허리를 숙이거나 바닥에 눕지 않고 차 밑 깊숙한 곳을 확인할 수 있습니다.

가구 · 가전 뒤쪽 확인

침대 밑, 가구 뒤, 세탁기 주변처럼 손과 눈이 닿지 않는 곳을 확인하세요.

운동 자세 확인

혼자 홈트할 때도 뒤모습과 전체 균형을 실시간으로 볼 수 있습니다.

옷 핏 확인

혼자서는 보기 어려운 뒷모습과 전체 핏을 한눈에 확인하세요.

수염 · 옆모습 확인

턱선, 구렛나루, 수염 라인까지 더 정확하게 볼 수 있습니다.

사각지대 확인

혼자 확인하기 어려운 위치도 다른 화면으로 선명하게 확인할 수 있습니다.

Local Connection

우리끼리만 안전하게,
로컬 연결

외부 서버를 거치지 않고 같은 Wi‑Fi 안에서 두 기기를 연결해 사용할 수 있도록 설계했습니다.

How To Use

사용 방법은 간단합니다

1

카메라용 휴대폰 준비

촬영할 휴대폰에서 연결용 QR 코드를 생성합니다.

2

Viewer용 휴대폰 준비

화면을 확인할 휴대폰에서 QR 스캔을 선택합니다.

3

QR 코드 스캔

카메라용 휴대폰에 표시된 QR 코드를 스캔합니다.

4

실시간 확인

연결이 완료되면 다른 휴대폰 화면에서 카메라 장면을 확인합니다.
